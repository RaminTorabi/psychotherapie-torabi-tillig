# Praxis Dr. med. Helene Torabi-Tillig

Website für die psychotherapeutische Praxis von Dr. med. Helene Torabi-Tillig in Berlin.

## GitHub Pages Setup

Diese Website nutzt Jekyll und GitHub Pages. 

### Aktivierung von GitHub Pages:

1. Repository auf GitHub erstellen
2. Dateien hochladen
3. In den Repository-Einstellungen unter "Pages" die Quelle auf "Deploy from a branch" setzen
4. Branch "main" und Ordner "/ (root)" auswählen
5. Website ist dann unter `https://[username].github.io/[repository-name]` erreichbar

### Anpassungen

Folgende Dateien sollten noch angepasst werden:

- `_config.yml`: E-Mail und Telefonnummer ergänzen
- `contact.md`: Kontaktdaten und Anfahrtsinformationen vervollständigen
- `about.md`: Qualifikationen und weitere Details hinzufügen
- `services.md`: Spezifische Behandlungsverfahren ergänzen

### Struktur

- `index.md` - Startseite
- `about.md` - Über mich
- `services.md` - Leistungen  
- `contact.md` - Kontakt & Anfahrt
- `impressum.md` - Impressum
- `datenschutz.md` - Datenschutzerklärung
- `_config.yml` - Jekyll-Konfiguration
